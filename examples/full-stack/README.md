# Full-Stack Project

This example demonstrates
- Creating a full-stack application with vite and vodka 
- Routes
- Controllers
- Using the api from frontend
- setting up and running project

## Steps to follow

**1. Initialize a new vite + vodka project**

Use the vodka CLI

```bash
vodka create my-app
```

Install frontend dependencies

```bash
cd my-app
cd frontend && npm install
cd ..
```

Start the application

```bash
vodka run dev
```

This starts both react application and go backend
